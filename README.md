## Design

A simple minimum viable product that traverses a path (like a linked list) or a graph (like a state machine). Most interactive nodes are question nodes. Question node contains a question and associated options for user to select from. Each option points to one or more other nodes which may also be an advice node. Traversal to next node is decided based on the option that is selected and the current state.

```tex
$$$

S_{t + 1} = function(S_t, x)

$$$

% where S_t is state at time t and x represents the option that is selected %

```
